Binance 자동 매매 시스템

선물 마켓 변동성에 대응하기 위한 헤지 전략 자동화 플랫폼의 아키텍처, 지표, 그리고 레퍼런스를 정리했습니다.

GitHub 레포지토리

핵심 매매 엔진과 FastAPI 백엔드를 공개한 레포지토리는 dslee1371/binance-futures-autotrader 에서 확인할 수 있습니다. 주요 브랜치에는 리스크 관리 모듈(risk)과 백테스트 파이프라인(backtest)이 분리되어 있습니다.

운영 성과 지표

  • 선물 포지션 급변 구간에서 자동 헤지로 미실현 손실을 평균 35% 완화 (2023년 6~9월 로그 기반).
  • Argo Workflows 기반 CI/CD 도입 후 전략 배포 리드타임 3시간 → 20분 단축.
  • Prometheus + Grafana 대시보드에서 P99 API 응답 시간 420ms → 210ms 개선.

발표 자료

FastAPIConf Seoul 2023 Lightning Talk에서 발표한 <Futures Risk Hedging with FastAPI> 슬라이드 PDF는 GitHub 발표 아카이브 에서 다운로드할 수 있습니다.
