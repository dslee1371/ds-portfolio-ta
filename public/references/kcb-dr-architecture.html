KCB DR(재해복구) 환경 구축

멀티 계정 AWS 인프라와 Kubernetes 기반 CI/CD 체계를 구축하면서 확보한 문서와 감사 로그를 제공합니다.

GitHub 레포지토리

IaC와 배포 파이프라인 템플릿은 dslee1371/kcb-dr-eks 에서 확인할 수 있습니다. Terraform 모듈과 Helm 차트가 분리되어 있어 재사용이 용이합니다.

아키텍처 개요

  • AWS Control Tower로 관리형 계정을 분리하고, 조직단위 SCP로 보안 정책을 중앙화.
  • EKS 기반 워크로드에 대해 Argo CD + Helm GitOps 전략을 도입하여 선언적 배포 구현.
  • Cross-Region S3 + RDS Aurora Global Database로 데이터 RPO를 5분 이내로 유지.
